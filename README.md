# CCDPHD-11-Dataset
It is a dataset that includes 11 classes of cotton images (10 diseases and pests, 1 healthy cotton). This dataset can be used for optimizing deep learning model for cotton diseases and pests classification.

The provided dataset is a part of our full dataset. Owing to the policy restrictions, we can not directly post the whole dataset here.
